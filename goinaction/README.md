## 第1章 Go语言开始

### 1.1 Go语言介绍
Go语言的出现，实则为并发而出现的语言也不过为。Java语言中利用多线程机制可以较为方便的实现程序并发，提高程序并发度。Go语言并发较于Java更加细化，在线程基础，利用goroutine再并发一次，粒度更小。

学习Go语言最重要的是学会写并发，也就是写goroutine和通道(channel)

### 1.2 Go语言环境配置
鄙人写Go语言是使用GoLand工具。推荐使用。GoLand安装和Go语言环境安装自行Google或百度即可。
